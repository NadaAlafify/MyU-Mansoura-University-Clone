# MyU - Mansoura University Portal Frontend Clone

##  Description
This project is a pixel-perfect, custom-built frontend clone of **MyU**, the official student services portal for Mansoura University. The core purpose of this challenge was to replicate a complex, dashboard-style platform layout using **Pure CSS Flexbox** without relying on any external CSS frameworks or UI kits.

It handles dynamic block alignments, centered typography, and an intuitive responsive structure for a seamless dashboard experience.

##  Live Demo
Explore the live web portal here: [https://nadaalafify.github.io/MyU/](https://nadaalafify.github.io/MyU/)

##  Tech Stack
- **HTML5:** Structuring the multi-grid dashboard components, system action triggers, and student portal navigation links.
- **CSS3 (Pure CSS):** Utilizing advanced **Flexbox layout properties** (`flex-wrap`, strict dimensioning, and directional alignment) to build a stable, symmetric dashboard grid.

##  Key Features & Technical Execution
- **Portal Dashboard Layout:** Replicating the exact 16-card grid interface containing vital university sections (e.g., Results, Personal Data, Academic Registration, and Credit Hour progress).
- **Flexbox Precision:** Flawless horizontal and vertical centering for nested custom icons and Arabic typography inside each individual card wrapper.
- **Symmetric Spacing:** Implementing balanced gutters and layouts to enhance the original platform's user experience.
- **100% Manual Execution:** Built entirely by hand to deeply master layout flow distribution and responsive container behaviors.
